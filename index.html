How to format the CSV

For now, we only support CSV files that look like:

Each row should be a set member, each column should be a set, and a non-zero number indicates that the row is a member of the column.

If you have ordered sets (such as in actual subway lines), the value can be used to indicate each row's position in its column set. For example, this file:
